# VB Code Vault 🚀

> **Vibe coding** > Live Preview: [codingkuh.my.id](https://codingkuh.my.id/)

Welcome to the **VB Code Vault**! This repository serves as a collection of static web pages and documentation templates written entirely in HTML. It is designed to be a quick reference and hosting repository for various coding standards, QA forms, and standard wordings.

## 📂 Repository Structure

This project consists of several HTML documents tailored for different purposes:

- `index.html` - The main landing page / entry point.
- `code.html` - Contains coding-related notes, snippets, or standards.
- `qa.html` - General Quality Assurance (QA) guidelines or forms.
- `qa-siemens.html` - Specific Quality Assurance documentation related to Siemens.
- `standard-wording.html` - A collection of standard phrases, wordings, or templates.
- `anmerkung.html` - Notes or annotations ("Anmerkung" in German).
- `CNAME` - Configuration file for mapping the custom domain (`codingkuh.my.id`).

## 🛠️ Built With

* **HTML5** (100%) - Pure, lightweight static markup.
